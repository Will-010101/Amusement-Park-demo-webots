# Amusement Park Simulation in Webots

## Overview

This project is a simple amusement park simulation developed using Webots.

The main objective was to design and simulate a Ferris wheel model, explore object modeling techniques, and implement rotational motion within the simulation environment.

---

## Project Information

- **Platform:** Webots
- **Programming Language:** Python
- **Project Type:** Robot & Environment Simulation

---

## Technologies

- Webots
- Python
- 3D Object Modeling
- Simulation

---

## Features

- Custom Ferris wheel model
- Rotational animation
- Motion simulation in Webots
- Educational robotics simulation
